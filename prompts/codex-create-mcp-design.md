# Prompt para diseñar herramientas MCP futuras

Tarea: diseñar posibles herramientas MCP para Granja Luna, sin implementarlas todavía.

Reglas:

- Diseñar primero herramientas de lectura y borrador.
- Evitar herramientas destructivas al inicio.
- Toda herramienta que escriba datos reales debe tener confirmación humana.
- Definir nombre, propósito, inputs, outputs, riesgos y errores esperados.

Prioridad inicial:

1. consultar memoria Markdown;
2. buscar ficha de medicamento;
3. preparar borrador de compra;
4. preparar borrador de venta;
5. preparar evento sanitario;
6. listar tareas pendientes;
7. resumir bitácora diaria.
