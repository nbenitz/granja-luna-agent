# Prompt para actualizar memoria

Tarea: incorporar nueva información operativa a la memoria Markdown de Granja Luna.

Reglas:

- No marcar nada como confirmado si Néstor no lo confirmó explícitamente.
- Si viene de conversación, usar `confirmed_by_user` solo cuando sea claro.
- Si hay dudas, usar `pending_review`.
- Si es una idea, usar `draft`.
- Si viene de `avicola-mbore`, usar `legacy_reference`.
- Registrar decisiones importantes en `memory/granja/decisiones.md`.
- No modificar fichas sanitarias sin indicar fuente.

Salida esperada:

1. Archivos que conviene modificar.
2. Diffs propuestos.
3. Preguntas pendientes.
4. Nivel de riesgo.
