# Workflow: Intake de tareas

## Objetivo

Convertir una entrada natural en una tarea clasificada y trazable.

## Pasos

1. Recibir entrada del usuario.
2. Identificar dominio probable.
3. Extraer datos detectados.
4. Listar datos faltantes.
5. Asignar nivel de riesgo.
6. Decidir si se responde, se pregunta o se propone registro.
7. Si corresponde, preparar contrato para el Orquestador.

## Salida mínima

- Resumen.
- Dominio.
- Datos detectados.
- Datos faltantes.
- Riesgo.
- Próxima acción.
- Confirmación requerida.
