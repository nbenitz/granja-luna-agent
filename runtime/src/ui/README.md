# UI Runtime

Estado: `draft`

La UI futura de Granja Luna debe consumir `ui_response` del runtime.

## Superficies posibles

- App propia de Granja Luna.
- App del Asistente Personal.
- CLI o consola.
- Futuro componente embebible.

## Regla

La experiencia puede verse distinta en cada superficie, pero la semantica debe ser la misma:

- campos;
- datos detectados;
- datos faltantes;
- riesgos;
- acciones;
- confirmacion requerida.

## No objetivo inicial

No crear todavia HTML/CSS arbitrario ni una app web completa.
