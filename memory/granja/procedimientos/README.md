# Procedimientos operativos

Guardar aquí procedimientos en evolución.

Ejemplos:

- mantenimiento de cama;
- limpieza y desinfección;
- ingreso de aves nuevas;
- manejo de pollitos;
- registro de compra;
- registro de venta;
- protocolo ante aves enfermas;
- incubación;
- inventario de depósito.

Cada procedimiento debe indicar estado, fecha, responsable y nivel de riesgo.
