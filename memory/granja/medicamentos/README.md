# Medicamentos y productos sanitarios

Este directorio guarda fichas Markdown de medicamentos, vacunas, vitaminas, desinfectantes y productos sanitarios usados o evaluados en Granja Luna.

## Regla sanitaria

Las fichas son memoria operativa y apoyo de organización. No sustituyen diagnóstico ni receta veterinaria.

## Template recomendado

Copiar `_template.md` para cada producto.
