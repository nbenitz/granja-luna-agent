# Contexto operativo de Granja Luna

Estado: `draft`

## Descripción

Granja Luna es una operación avícola y agropecuaria en evolución. El objetivo del subagente es aprender progresivamente cómo se gestiona la granja y ayudar a registrar, ordenar y automatizar actividades.

## Dominios iniciales

- Sanidad
- Alimentación
- Stock e insumos
- Compras
- Ventas
- Incubación
- Reproductores
- Infraestructura
- Tareas y mantenimiento
- Reportes

## Criterio de aprendizaje

Cada actividad real puede convertirse en:

1. entrada de bitácora;
2. tarea;
3. procedimiento;
4. ficha de conocimiento;
5. registro estructurado futuro;
6. herramienta/API/MCP si se repite y requiere precisión.
