# Taxonomía de estados

Usar estos estados en memoria y documentos:

- `confirmed`: confirmado y estable.
- `confirmed_by_user`: confirmado explícitamente por Néstor.
- `pending_review`: útil pero pendiente de validar.
- `draft`: borrador.
- `legacy_reference`: tomado de un sistema/documento histórico; no asumir como verdad actual.
- `archived`: conservado por historia.

Evitar mezclar estados en una misma afirmación. Si un documento tiene datos mixtos, cada sección debe aclarar su estado.
