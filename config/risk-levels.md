# Configuración de niveles de riesgo

| Nivel | Descripción | Confirmación |
|---|---|---|
| bajo | Organización, resumen, borrador o lectura | No siempre |
| medio | Puede afectar datos operativos | Sí antes de registrar |
| alto | Puede afectar dinero, sanidad, stock o decisiones importantes | Sí explícita |
| critico | Puede afectar vida animal, seguridad, dispositivos físicos o datos sensibles | No ejecutar automáticamente |

## Dominios con riesgo alto por defecto

- Sanidad
- Tratamientos
- Ajustes de stock
- Automatización IoT con actuadores
- Eliminaciones
- Confirmación de compras/ventas
